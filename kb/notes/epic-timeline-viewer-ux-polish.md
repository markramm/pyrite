## Overview

UX evaluation of the capturecascade.org timeline viewer identified 8 issues and 4 design recommendations. The viewer is functional and well-structured but needs polish on information density, view coordination, and first-time user experience.

## Priority Fixes
1. Default D3 view to 2020s instead of full 900-year range
2. Add importance indicator to table rows
3. Add date range filter to table view

## Bugs/Issues
4. No state sharing between table and D3 views
5. Slide-out panel body text rendering is flat (no lists, blockquotes)
6. Search doesn't cover body text (lightweight index limitation)
7. No date range filter in table
8. External link icon column adds clutter

## Design Improvements
9. Add compact stats header to viewer
10. Row hover preview (body text snippet)
11. Active filter chip bar above table
12. D3 category legend
