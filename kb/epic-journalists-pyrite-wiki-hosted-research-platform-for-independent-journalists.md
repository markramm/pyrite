## Vision

An invitation-only hosted Pyrite instance (journalists.pyrite.wiki) pre-loaded with curated investigative research KBs, configured for multi-user access with BYOK (bring your own key) LLM integration.

Three access paths:
1. **Web UI** -- browse KBs, run research workflows, write findings
2. **Authenticated MCP endpoint** -- connect from Claude Code or Google CLI
3. **REST API** -- custom tooling and integrations

## Pre-Populated KB Library

Six curated, cross-linked investigative KBs:
- **Capture Timeline** -- 4800+ chronological institutional capture events
- **Capture Research** -- deep analysis of capture mechanisms and patterns
- **Solidarity Research** -- American resistance infrastructure, historical and current
- **Thiel Network** -- influence and funding mapping (21+ companies, $280B+ contracts)
- **Epstein Network** -- connections, compromise patterns
- **Surveillance Industrial Complex** -- surveillance state apparatus and technology

Cross-KB links surface connections across domains (e.g., ICE detention -> SIC -> Thiel/Palantir -> Capture Timeline).

## Key Differentiator

Not selling software (it is open source). Selling a research head start -- curated, sourced, cross-linked investigative knowledge that took months to build, plus infrastructure to extend it with agent workflows.

Like Aleph but without the infrastructure bloat. OSINT pipelines feed Pyrite KBs; Pyrite is the knowledge layer, not the collection layer.

## Success Criteria

- Invitation-only instance running with multi-user auth
- 6 pre-populated KBs accessible to invited journalists
- BYOK API key integration for AI features
- Secure authenticated MCP endpoint for CLI users
- At least one journalist actively using it for published investigation
