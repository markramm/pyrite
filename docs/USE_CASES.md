
# Use Cases & Journeys

## 1) Processing an Article
1. Create a seed note from key idea(s) of an article.
2. Run CEQRC to simplify and crystallize understanding.
3. Connect to related notes; store the source as a link.

## 2) Developing a Fleeting Idea
1. Capture a quick draft.
2. Use probe to surface gaps; refine the draft.
3. Title, tag, and connect to existing concepts.

## 3) Synthesizing Existing Notes
1. Identify two related notes.
2. Create a synthesis note that **extends** or **refines** them.
3. Link them using typed relations.
